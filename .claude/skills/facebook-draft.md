# Facebook下書き生成スキル

こぼり良江（小堀良江）栃木市議会議員のFacebook投稿案を、活動メモから生成します。

## 使い方

```
/facebook-draft ideas/YYYYMMDD_トピック.md
```

例: `/facebook-draft ideas/20260328_nursery_visit.md`

---

## Claudeへの指示

このスキルが呼び出されたら、以下の手順を実行してください。

### Step 1: ファイルを読み込む

1. 引数で指定された `ideas/` のファイルを Read ツールで読み込む
2. `checks/forbidden_words.md` を読み込む（安全チェック用）

### Step 2: 下書きを生成する

以下の仕様でFacebook投稿案を書く。

**文体・トーン**:
- 信頼感・誠実さ・地域リーダーとしての言葉
- 丁寧語・ですます調
- 政策背景・課題の説明もOK（Facebookは長文が読まれやすい）
- 感情的になりすぎず、落ち着いたトーンで
- 絵文字は使わないか最小限（信頼感重視）

**本文の長さ**: 200〜500文字（フレキシブル）

**構成の例**:
```
（冒頭: 活動の概要を1〜2文で）

（本文: 活動の内容・背景・政策との関連）

（結び: 今後の取り組みや市民へのメッセージ）
```

**段落分け**:
- 1段落3〜5文程度
- PCでも読みやすいよう段落間に空行を入れる

**ハッシュタグ**:
- 0〜5個（少なめが効果的）
- 投稿末尾にまとめて置く
- 必要なければ省略してもOK
- 使う場合は `#栃木市` `#こぼり良江` を優先

### Step 3: 出力ファイル名を決める

- 例: `ideas/20260328_nursery_visit.md` → `drafts/facebook/20260328_nursery_visit.md`

### Step 4: 安全メモを書く

以下の観点で `安全メモ` を記入する:
- 現在の期間モード（今日の日付から判断）
- CATEGORY_Aの禁止語が含まれていないか
- CATEGORY_Bの要注意語が含まれていないか
- 問題なければ「普段の議員活動報告として問題なし」と明記
- Facebookは知人・支持者が多く読むため、記述内容の正確性も確認を促す

### Step 5: ファイルを保存する

```markdown
---
date: YYYY-MM-DD
platform: facebook
idea_source: ideas/FILENAME.md
status: draft
---

## 要約
（1〜2行で何についての投稿か）

## 本文
（Facebook投稿文。段落分けで読みやすく）

## ハッシュタグ候補
#栃木市 #こぼり良江 （使う場合のみ記載）

## 画像案
（添付する場合は説明。Facebookはアルバム形式も可能）

## 安全メモ
⚠️ 公選法チェック: （問題なし、または具体的な注意点）
```

### Step 6: ユーザーに報告する

保存完了後、以下を伝える:
- 保存先ファイルパス
- 本文の文字数
- 安全メモの要点
- Facebookは公開範囲（全体公開 / 友達のみ）を確認するよう促す

---

## 禁止事項

- `status: posted` をセットしない
- 安全メモを省略しない
- 投票依頼・支持依頼の表現を生成しない（告示前）
- 個人の住所・電話番号等の個人情報を含めない
