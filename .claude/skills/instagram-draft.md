# Instagram下書き生成スキル

こぼり良江（小堀良江）栃木市議会議員のInstagram投稿案を、活動メモから生成します。

## 使い方

```
/instagram-draft ideas/YYYYMMDD_トピック.md
```

例: `/instagram-draft ideas/20260328_nursery_visit.md`

---

## Claudeへの指示

このスキルが呼び出されたら、以下の手順を実行してください。

### Step 1: ファイルを読み込む

1. 引数で指定された `ideas/` のファイルを Read ツールで読み込む
2. `templates/post_structure.md` を読み込む（フォーマット確認）
3. `checks/forbidden_words.md` を読み込む（安全チェック用）

### Step 2: 下書きを生成する

以下の仕様でInstagram投稿案を書く。

**文体・トーン**:
- 親しみやすく、温かく、市民目線
- 体言止め・感嘆表現OK（「感動しました！」「すごいですね✨」など）
- 絵文字を適度に使う（多すぎない・1〜3個/段落が目安）
- 一人称は「私」または「こぼり良江」
- 体験談・感想を具体的に盛り込む

**本文の長さ**: 300〜800文字（絵文字・改行含まず）

**構成の例**:
```
（出来事・場面の描写）

（感じたこと・気づき）

（政策への繋がりや今後の取り組み）

（締めの一言）
```

**改行**: 2〜3行ごとに空行を入れてモバイルで読みやすくする

**ハッシュタグ**:
- 15〜25個
- 最初の3つは必ず: `#栃木市 #こぼり良江 #小堀良江`
- 活動内容・政策テーマ・地域名に関連するタグを追加
- `templates/hashtag_bank.md` を参照して適切なタグを選ぶ

### Step 3: 出力ファイル名を決める

- アイデアファイルの日付とトピックからスラグを作る
- 例: `ideas/20260328_nursery_visit.md` → `drafts/instagram/20260328_nursery_visit.md`

### Step 4: 安全メモを書く

以下の観点で `安全メモ` を記入する:
- 現在の期間モード（今日の日付から判断: 〜4/11=事前運動禁止、4/12〜4/18=選挙運動、4/19〜=通常）
- CATEGORY_Aの禁止語が含まれていないか
- CATEGORY_Bの要注意語が含まれていないか
- 問題なければ「普段の議員活動報告として問題なし」と明記

### Step 5: ファイルを保存する

以下のフォーマットで `drafts/instagram/` に保存する:

```markdown
---
date: YYYY-MM-DD
platform: instagram
idea_source: ideas/FILENAME.md
status: draft
---

## 要約
（1〜2行で何についての投稿か）

## 本文
（Instagram投稿文。絵文字・改行含む）

## ハッシュタグ候補
#栃木市 #こぼり良江 #小堀良江 #...

## 画像案
（どんな写真・画像を添付するか。具体的なシーンや構図を提案）

## 安全メモ
⚠️ 公選法チェック: （問題なし、または具体的な注意点）
```

### Step 6: ユーザーに報告する

保存完了後、以下を伝える:
- 保存先ファイルパス
- 本文の文字数
- ハッシュタグ数
- 安全メモの要点
- 「hookが自動チェックを実行しました。チェック結果セクションをご確認ください。」

---

## 禁止事項

- `status: posted` をセットしない
- 安全メモを省略しない
- 投票依頼・支持依頼の表現を生成しない（告示前）
- 名前の誤表記を使わない
