# 週次投稿プラン生成スキル

今週のideasフォルダの活動メモを確認し、投稿カレンダーを作成します。

## 使い方

```
/weekly-summary
```

（引数なし。または `/weekly-summary 2026-03-23` のように週の始まりの日付を指定可）

---

## Claudeへの指示

このスキルが呼び出されたら、以下の手順を実行してください。

### Step 1: ideas/ のファイルを確認する

1. `ideas/` フォルダ以下の `.md` ファイルを全件列挙する
2. TEMPLATE_idea.md はスキップ
3. ファイル名の日付（YYYYMMDD）でソートする
4. 直近2週間分（今週＋先週）のファイルを対象とする

### Step 2: 各ファイルの状況を確認する

各アイデアファイルについて、対応する下書きが作成済みかチェックする:
- `drafts/instagram/` に対応ファイルがあるか
- `drafts/x/` に対応ファイルがあるか
- `drafts/facebook/` に対応ファイルがあるか

### Step 3: 週次サマリーを生成する

以下のフォーマットでサマリーを出力する（ファイル保存は不要、Claudeの返答として出力）:

```
## 週次投稿プラン（YYYY年M月W週）

### 今週のアイデア一覧

| 日付 | トピック | Instagram | X | Facebook | 備考 |
|------|---------|-----------|---|---------|------|
| MM/DD | トピック名 | ✅ 作成済 | ⬜ 未作成 | ✅ 作成済 | |
| MM/DD | トピック名 | ⬜ 未作成 | ⬜ 未作成 | ⬜ 未作成 | |

### 未作成の下書き

以下のコマンドで下書きを生成できます:
/instagram-draft ideas/YYYYMMDD_XXX.md
/x-draft ideas/YYYYMMDD_XXX.md
/facebook-draft ideas/YYYYMMDD_XXX.md

### 下書き済み・承認待ち

| ファイル | ステータス | 作成日 |
|---------|-----------|-------|
| drafts/instagram/XXXXXX.md | draft | MM/DD |

### 今週の投稿スケジュール案

（アイデアの内容・重要度・プラットフォーム特性に基づいて提案）

推奨投稿タイミング:
- Instagram: 火曜・金曜の午前中（エンゲージメント高め）
- X: イベント・会議の当日か翌日
- Facebook: 週に1〜2回、月曜か木曜

| 投稿予定日 | プラットフォーム | ネタ | 下書き |
|-----------|---------------|-----|-------|
| MM/DD（火） | Instagram | ○○ | drafts/instagram/XXX.md |
| MM/DD（水） | X | ○○ | 未作成 |

### アクション項目

□ 〇〇の下書きを生成する
□ 〇〇の下書きをレビューして承認する
□ 〇〇をInstagramに投稿する（手動）
```

### Step 4: 現在の選挙モードを確認・表示

今日の日付から選挙モードを判定して先頭に表示する:

```
📅 現在のモード: 【事前運動禁止期間】（〜2026/4/11）
⚠️ 投票依頼・支持依頼の表現は使用不可。普段の活動報告のみ。
```

または

```
📅 現在のモード: 【選挙運動期間】（2026/4/12〜4/18）
✅ 投票呼びかけ表現OK。積極的に発信してください。
```

---

## 補足

- このスキルはファイルを生成しません（計画を出力するだけ）
- 実際の下書き生成は `/instagram-draft`、`/x-draft`、`/facebook-draft` を使ってください
- `status: approved` になっているファイルは投稿準備完了として表示します
