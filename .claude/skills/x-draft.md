# X（Twitter）下書き生成スキル

こぼり良江（小堀良江）栃木市議会議員のX投稿案を、活動メモから生成します。

## 使い方

```
/x-draft ideas/YYYYMMDD_トピック.md
```

例: `/x-draft ideas/20260328_nursery_visit.md`

---

## Claudeへの指示

このスキルが呼び出されたら、以下の手順を実行してください。

### Step 1: ファイルを読み込む

1. 引数で指定された `ideas/` のファイルを Read ツールで読み込む
2. `checks/forbidden_words.md` を読み込む（安全チェック用）

### Step 2: 下書きを生成する

以下の仕様でX投稿案を書く。

**文体・トーン**:
- やや硬め・簡潔・事実ベース
- 敬語・報告調（「〜しました」「〜です」）
- 絵文字は最小限（1投稿につき0〜2個まで）
- 情報を詰め込みすぎず、1つのことを端的に伝える

**⚠️ 最重要: 140文字制限**:
- 本文＋ハッシュタグを合わせて **140文字以内**
- URLを含む場合は **23文字** として換算する
- 全角1文字 = 1文字、半角1文字 = 1文字（Xの仕様）
- 文字数を確認してから保存すること
- 140文字を超える場合は「スレッド形式」で複数投稿案に分割する

**スレッド形式**（内容が長い場合）:
```
1/2
（1つ目のツイート本文 ＋ ハッシュタグ）

2/2
（2つ目のツイート本文）
```

**ハッシュタグ**:
- 1〜3個（文末に置く）
- 必ず `#栃木市` と `#こぼり良江` を含める
- ハッシュタグも140文字のカウントに含まれることに注意

### Step 3: 出力ファイル名を決める

- 例: `ideas/20260328_nursery_visit.md` → `drafts/x/20260328_nursery_visit.md`

### Step 4: 安全メモを書く

以下の観点で `安全メモ` を記入する:
- 現在の期間モード（今日の日付から判断）
- CATEGORY_Aの禁止語が含まれていないか
- 問題なければ「普段の議員活動報告として問題なし」と明記
- 文字数も記録しておく

### Step 5: ファイルを保存する

```markdown
---
date: YYYY-MM-DD
platform: x
idea_source: ideas/FILENAME.md
status: draft
char_count: N
---

## 要約
（1行で何についての投稿か）

## 本文
（X投稿文。140文字以内。ハッシュタグ含む）

（スレッドの場合は「1/2」「2/2」形式で複数案を並べる）

## ハッシュタグ候補
#栃木市 #こぼり良江 （本文末尾に含めたものを記載）

## 画像案
（添付する場合は説明。Xは画像ありでエンゲージメント向上）

## 安全メモ
⚠️ 公選法チェック: （問題なし、または具体的な注意点）
文字数: N文字（140文字以内 ✅ / オーバー ❌）
```

### Step 6: ユーザーに報告する

保存完了後、以下を伝える:
- 保存先ファイルパス
- **正確な文字数**（140文字制限の確認のため必ず伝える）
- スレッドの場合は投稿数
- 安全メモの要点

---

## 禁止事項

- 140文字を超えた本文を保存しない（スレッド形式に分割すること）
- `status: posted` をセットしない
- 安全メモを省略しない
- 投票依頼・支持依頼の表現を生成しない（告示前）
