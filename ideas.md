# 衆議院小選挙区クイズ Web版 — デザインブレインストーミング

## 概要
第51回衆議院議員総選挙の小選挙区当選者暗記ゲーム（4択クイズ）のWebアプリ版。
ターゲット：政治に関心のある学生・若者。モバイルファーストだがデスクトップでも快適に使えること。

## Idea 1: 「和モダン × 新聞活版」スタイル

### Design Movement
和モダニズム + エディトリアルデザイン。新聞の活版印刷のような質感と、日本の伝統的な色彩を現代的に再解釈。

### Core Principles
1. **活版印刷のテクスチャ感** — 紙の質感、インクの滲み、縦書きアクセント
2. **余白の美学** — 日本的な「間」を活かした大胆な余白
3. **情報の階層化** — 新聞のように見出し→本文→注釈の明確な階層
4. **朱色と墨色の対比** — 日本の公文書・印鑑を想起させるカラーリング

### Color Philosophy
- 墨色（#1a1a2e）: 権威と信頼を表す基調色
- 朱色（#c0392b）: 正解・当選を示すアクセント、印鑑のイメージ
- 和紙白（#f5f0e8）: 温かみのある背景
- 藍色（#2c3e6b）: 情報の補助色
- 感情：「格式」「信頼」「知的好奇心」

### Layout Paradigm
縦書きタイトル + 横書き本文のハイブリッド。カード型UIだが角を丸めず直線的。新聞のカラム構造を意識した非対称グリッド。

### Signature Elements
1. 朱色の丸印（○）アニメーション — 正解時に印鑑を押すようなスタンプエフェクト
2. 縦書きの選挙区名表示
3. 紙のテクスチャを持つカード背景

### Interaction Philosophy
「一問一答の緊張感」— 新聞を読むような静的な美しさの中に、回答時のダイナミックなフィードバック。

### Animation
- 問題カードのスライドイン（横からフェード）
- 正解時：朱色の○が押印されるようなスケールアニメーション
- 不正解時：墨色の×がにじむように表示
- ページ遷移：和紙がめくれるようなトランジション

### Typography System
- 見出し: Noto Serif JP (Bold) — 格式と伝統
- 本文: Noto Sans JP (Regular) — 可読性
- 数字・データ: Tabular Lining のモノスペース風
</idea>

## Idea 2: 「選挙ボード × データダッシュボード」スタイル

### Design Movement
インフォグラフィック・ブルータリズム。選挙速報番組のテロップやデータ表示を模したUI。

### Core Principles
1. **速報感** — テレビの選挙特番のようなリアルタイム感
2. **データの可視化** — 得票数をバーチャートで即座に表示
3. **コントラストの強さ** — 濃紺背景に白文字、結果は色分け
4. **タイル型レイアウト** — ダッシュボードのようなモジュール構成

### Color Philosophy
- 濃紺（#0a1628）: テレビ局の選挙速報背景
- 白（#ffffff）: テロップの文字色
- 赤（#e74c3c）: 与党系・当選表示
- 青（#3498db）: 野党系
- 黄（#f1c40f）: ハイライト・スコア
- 感情：「興奮」「速報」「データドリブン」

### Layout Paradigm
フルスクリーンダッシュボード。上部にスコアバー、中央に問題、下部に4択ボタン。結果画面はデータテーブル+ミニバーチャート。

### Signature Elements
1. テレビテロップ風の選挙区名表示（下線付きスライドイン）
2. 得票数のアニメーションカウントアップ
3. 当確ランプの点灯アニメーション

### Interaction Philosophy
「選挙速報を自分で体験する」— テレビの選挙特番を操作している感覚。

### Animation
- テロップのスライドイン（左から右へ）
- 当確時：赤い「当」バッジがフラッシュ
- スコアのカウントアップアニメーション
- 選択肢のホバー時にグロー効果

### Typography System
- 見出し: Noto Sans JP (Black) — テロップの力強さ
- データ: Roboto Mono — 数値の整列
- 本文: Noto Sans JP (Medium)
</idea>

## Idea 3: 「ミニマル・カードゲーム」スタイル

### Design Movement
スカンジナビアン・ミニマリズム × ゲーミフィケーション。カードゲームのような楽しさをクリーンなデザインで。

### Core Principles
1. **遊び心のあるミニマル** — 余計な装飾を排除しつつ、ゲームとしての楽しさを演出
2. **カードメタファー** — 選択肢をトランプカードのように扱う
3. **マイクロインタラクション** — 小さな動きで大きな満足感
4. **プログレッション感** — 進捗が視覚的に分かるUI

### Color Philosophy
- オフホワイト（#fafaf9）: 清潔感のある背景
- チャコール（#292524）: 読みやすいテキスト
- インディゴ（#4338ca）: プライマリアクション
- エメラルド（#059669）: 正解フィードバック
- ローズ（#e11d48）: 不正解フィードバック
- 感情：「軽快」「集中」「達成感」

### Layout Paradigm
シングルカラム・カード中心。モバイルファーストで、デスクトップでは最大幅480pxのセンターカード。余白を大胆に使い、一度に見せる情報を最小限に。

### Signature Elements
1. カードのフリップアニメーション（正解発表時）
2. 連続正解時のストリークカウンター（炎エフェクト）
3. プログレスドットインジケーター

### Interaction Philosophy
「一問に集中する」— 画面上の情報を最小限にし、回答に集中できる環境。

### Animation
- 選択肢カードの3Dフリップ
- 正解時：カードが緑に光り、チェックマークがバウンスイン
- 不正解時：カードが赤く震える（シェイク）
- 画面遷移：カードがスタックから引き出されるような動き

### Typography System
- 見出し: Inter (Bold) + letter-spacing: -0.02em — モダンで引き締まった印象
- 本文: Inter (Regular) — 高い可読性
- 数字: Inter (Semibold) Tabular — データの整列
</idea>

<text>カードゲームのメタファーを活かしたミニマルデザイン。遊び心とクリーンさの両立。</text>

## 選択: Idea 2 「選挙ボード × データダッシュボード」

理由：
- 選挙クイズというテーマに最もマッチする「選挙速報」の世界観
- データの可視化（得票数バーチャート）が結果画面の情報密度を高める
- 濃紺背景のダークテーマは長時間のクイズプレイに目に優しい
- テロップ風アニメーションが「次の問題へ」のテンポ感を演出
- 政治に関心のあるターゲット層に刺さるビジュアル
