## 公職選挙法について（背景知識）

公職選挙法第129条は「選挙運動は、各選挙につき候補者の届出のあった日から選挙の期日の前日まで」と定めています。
つまり告示日（2026年4月12日）より前に、投票を呼びかけたり支持を求めたりする行為（事前運動）は禁止されています。

違反した場合は同法第239条により処罰の対象となります（1年以下の禁固または30万円以下の罰金）。

## CATEGORY_A: 絶対禁止（告示前は使用不可）

告示前（〜2026/4/11）にこれらの表現が含まれていた場合、**ERROR** として検出します。
告示後（2026/4/12〜4/18）は選挙運動として合法ですが、告示前の投稿に混入しないよう注意。

```
支持をお願い
支持してください

一票をお願い
一票よろしく
○○に一票
こぼり良江に一票
小堀良江に一票
よろしく一票
```

## CATEGORY_B: 要注意（文脈次第でNG・WARNINGとして検出）

単体では問題ないが、選挙に向けた文脈・選挙直前期に使用すると事前運動と受け取られる恐れがある表現。
告示前は特に慎重に使用してください。**WARNING** として検出します。

```
応援してください
応援よろしくお願いします
力添えをお願い
お力添えを
皆さまのご支援を
ご支援をお願い
拡散をお願いします
シェアをお願いします
フォローをお願いします
選挙に向けて
次の選挙では
選挙活動として
候補者として
頑張ります（選挙文脈で使う場合）
```

## CATEGORY_C: 名前表記エラー（ERROR として検出）

以下の表記が含まれている場合、**ERROR** として検出し修正を促します。

```
こぼりよしえ
小堀よしえ
こぼり良枝
小堀良枝
コボリ良江
コボリよしえ
こぼりヨシエ
小堀ヨシエ
```

正しい表記:
- こぼり良江
- 小堀良江
- Kobori Yoshie（英語文脈のみ）

## 安全な表現の例（告示前でもOK）

```
本日、〇〇委員会に出席しました
〇〇の現場を視察してきました
栃木市の〇〇課題について、議会で取り上げています
市民の皆さんからご意見をいただきました
〇〇イベントにお邪魔しました
議会報告です
```

これらは「普段の議員活動報告」であり、選挙運動ではありません。

## 判断が迷う場合のフローチャート

```
投稿しようとしている内容は...

「日常の活動報告・政策説明」
    ↓ YES → OK（ただしCATEGORY_Aの表現を使わないこと）

「有権者に何かを求めている」
    ↓ YES → 何を求めているか？
        「投票・支持・当選」に関係する → 絶対NG（告示前）
        「意見・参加・問い合わせ」に関係する → OK
        「拡散・フォロー・応援」に関係する → 文脈次第・要注意
```
